# v0.19.1 — fix: KB sync on content-only installs

Patch for the maria UAT rollout of [v0.19.0](v0.19.0.md).

### Fixes

- **KB wiki sync failed on every image-baked KB** (`Not a git repository: /data/knowledge/<label>`). gbrain's sync uses git commits as its incremental checkpoint, but KBs installed via the Docker bake / `slaude install` copy ship content without `.git`. `syncKbWikis` now self-initializes a local checkpoint repo (never pushed; exists purely so sync can diff between runs) and commits drift before each sync. (`src/knowledge/brain-sync.ts`)
