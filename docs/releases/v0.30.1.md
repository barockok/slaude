## v0.30.1 — first release with baremetal installer assets

This is the first tag cut *after* the baremetal installer + self-update landed
(#63). Its purpose is operational: produce the release artifacts the installer
consumes — a curated source tarball plus `sha256sums.txt` attached to the GitHub
release — so `curl … | bash` installs and `slaude update` work end-to-end. No
runtime behavior changes from v0.30.0.

### Internal

- **Release now ships installer artifacts.** `release.yml` runs
  `scripts/package-release.sh` and attaches `slaude-0.30.1.tar.gz` +
  `sha256sums.txt` to the release. Earlier tags (≤ v0.30.0) predate the installer
  and carry no such assets, so the baremetal `install.sh` / `slaude update` paths
  resolve to **v0.30.1 or newer**. See
  `docs/findings/2026-06-22-baremetal-installer.md`.

- **Version bump only** — `package.json` → `0.30.1`. The tarball name derives from
  this version, so the tag (`v0.30.1`) and `package.json` version are kept in lock-step.
