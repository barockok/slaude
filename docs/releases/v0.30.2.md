## v0.30.2 — clean installer-asset release

Supersedes the never-completed v0.30.1. The v0.30.1 tag name was permanently
reserved by GitHub release immutability (a published immutable release burns the
tag name for good, even after the release is deleted), so this is the first
*clean-history* tag to ship the baremetal installer artifacts. No runtime
behavior changes from v0.30.0.

### Internal

- **Release ships installer artifacts.** `release.yml` runs
  `scripts/package-release.sh` and attaches `slaude-0.30.2.tar.gz` +
  `sha256sums.txt`. The baremetal `install.sh` / `slaude update` paths resolve to
  **v0.30.2 or newer** (tags ≤ v0.30.0 predate the installer and carry no assets;
  v0.30.1 was abandoned). See `docs/findings/2026-06-22-baremetal-installer.md`.

- **Commit-author hygiene.** History no longer carries AI co-author trailers; a
  `.githooks/commit-msg` hook now rejects them (enable per clone with
  `git config core.hooksPath .githooks`).

- **Version bump only** — `package.json` → `0.30.2`, kept in lock-step with the tag.
